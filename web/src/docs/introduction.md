Welcome to **PKG**, we tracks software releases so you don’t have to. We monitor upstream sources for new versions and notify you as soon as they are available.

---

## How it Works

We monitor official repositories, websites, and APIs for the software your team manages. When a change is detected, we capture the version number, release date and whether the update is a security patch or a breaking change.

### Features
- Receive a notification the moment a new version is detected.

## Quick Start

Getting started with PKG takes less than a minute:

1. **Explore:** Browse the [App Repository](https://pkg.rewake.org/apps) to see if the software your organization manages is listed.
2. **Authenticate:** [Sign in](https://pkg.rewake.org/notifications) using GitHub or a Magic Link.
3. **Subscribe:** Click the **Bell Icon** next to any application to start receiving notifications.

## Contributing

PKG is only as strong as its registry. If you notice a missing application or incorrect versioning data, please consider contributing to the tracking definitions on our [GitHub repository](https://github.com/mozzo1000/pkg).
