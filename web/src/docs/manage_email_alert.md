This guide explains how to use the PKG notification system to receive automated updates when new software versions are detected.

## The Bell Subscription System

PKG uses a "Bell" icon system to manage your subscriptions. This allows for granular control over which applications trigger an email alert.

### Subscribing to an Application

1. **Navigate to Account Settings**: Ensure you are logged in and viewing the Account Settings page.
2. **Locate the Application**: Scroll through the grid of tracked applications or use your browser's search function (Ctrl+F) to find a specific application.
3. **Toggle the Bell**: Click the card of the application you wish to follow.
* When the bell icon is **solid and highlighted**, you are successfully subscribed.
* The application card will change its border color to indicate an active subscription state.

### Unsubscribing

To stop receiving updates for a specific application, click the application card again. The bell icon will return to a faded, outlined state, and you will no longer receive alerts for that specific application.

---

## Digest Email

To prevent inbox clutter, PKG bundles all version updates detected within a 24-hour period into a single email.

### Identifying the Sender

All notification emails are dispatched from a dedicated address. To ensure these alerts are not filtered into your junk folder, add the following address to your contacts or safe-sender list:

`notifications@pkg.rewake.org`


Version signals are processed continuously, but the email dispatch occurs **once per day**.
